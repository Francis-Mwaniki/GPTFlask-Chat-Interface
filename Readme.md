# GET STARTED 

## Using OpenAI GPT-3 API
### 1. Create an account on OpenAI
### 2. Get your API key
### 3. Paste your API key in app.py


## 1. Install
```bash
 pip install Flask jsonify requests openai
```
## 2. Run
```bash
python app.py
```
